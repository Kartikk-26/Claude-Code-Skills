## Technical Specifications

### Recommended Sizes
- Full width: 1920x1080 (HD), 2560x1440 (2K)
- Pattern tile: 512x512 (seamless)
- Section background: 1920x600

### Format Recommendations
- Gradients: SVG or CSS (not image)
- Patterns: PNG (transparent) or SVG
- Complex: WebP or optimized JPG

### Performance Tips
1. Use CSS gradients when possible
2. Compress images (WebP preferred)
3. Consider lazy loading for below-fold
4. Use appropriate resolution (not oversized)
5. Test on different screen sizes
